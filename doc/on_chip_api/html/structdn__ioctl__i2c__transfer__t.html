Detailed Description

Specify what to transfer.

Pass (a pointer to) this structure in the args parameter of dn_ioctl(), when setting request to DN_IOCTL_I2C_TRANSFER.

Field Documentation

slaveAddress

Address of I2C slave device to communicate with.

This is the 7-bit address of the device, i.e.

its value can not be larger than DN_I2C_MAX_7BIT_ADDR.

writeBuf

Pointer to the buffer holding the data to write over I2C.

Set this element NULL when reading from an I2C device.

readBuf

Pointer to the buffer which will be filled by data read over I2C.

Set this element NULL when transmitting to an I2C device.

Clear the following bytes of this buffer. The driver will set those to the value from the slave device over the I2C bus.

writeLen

Number of bytes to write to the I2C slave.

Set this element 0 when reading from an I2C device.

The length of the dn_ioctl_i2c_transfer_t::writeBuf buffer.

readLen

Number of bytes to read from the I2C slave.

Set this element 0 when transmitting to an I2C device.

The length of the dn_ioctl_i2c_transfer_t::readBuf buffer.

timeout

Timeout.

Timeout, in milliseconds, to wait for the I2C transfer to complete.
